What this is base completer

This is a base model, not a chatbot. It was trained on next-token prediction only, so it continues text rather than answering questions. Prompt it with the opening of a sentence and watch it complete the thought.

The honest quality metric is held-out validation perplexity: 8.50 (lower is better). It speaks the legal register fluently (case-citation phrasing, procedural language) but it does not know facts, at 125M parameters a model holds only about 31MB of usable knowledge. Grounded facts would need retrieval (RAG).
